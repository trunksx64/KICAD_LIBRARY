Librerías para KiCAD por xDNA Electronics &amp; Design

# WARNING AND DISCLAIMER!!!

Estas son librerías para KiCad 10 que han sido creadas, modificadas y/o diseñadas por mi parte para la ejecución de proyectos tanto privados o personales, utilizando Freecad para Crear desde cero los modelos 3D de algunos componentes que las Librerías principales de KiCad no posee por algún motivo en específico. Por lo cual pueden diferir del modelo original u Oficial del creador, el Footprint y Symbol están totalmente producidos y basados a partir de la hoja de Datos de dicho componente, lo cual Garantiza su uso, pero tampoco pueden estar libre de errores.

De igual Manera estas librerías no emplean el formato de nombres o rutas estandarizados para el EDA KiCad o programas similares, a esto no espere nomenclaturas oficiales o estandarizadas.

Use estas librerías bajo su responsabilidad con o sin alguna garantía por mi parte, siéntase libre de modificarlas o distribuirlas, solo pido que se haga reconocimiento a mi creación en caso de usarlas para algún proyecto o diseño.

No todos los Diseños de los modelos 3D están disponibles, debido a la complejidad y tiempo que requieren, por lo que estos ha sido tomados de las web de mismo fabricante y adaptados a las necesidades de los desarrollos. no espere el archivo original para su modificación.
